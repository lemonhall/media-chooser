MediaChooserby Chute

  • Collect media from users with just a few lines of code.
  • Allow selecting from Facebook, Picasa, Flickr and Instagram.
  • Or let user upload media from his computer.

Get started or take a demo.

  1. Register for an account.

    2. We'll email your registration data.

  2. Setup MediaChooser.

    3. Include it into your page first: 

    <script src="http://cdn.getchute.com/media-chooser-0.1.1.min.js"></script>
    Note: jQuery is required.
    Then set your application identifier, which you received in an email: 
    Chute.setApp('your identifier');

  3. Start collecting media.

    4. Chute.MediaChooser.choose(function(urls, data){
	// urls is an array of direct media URLs
	// data array contains additional information about media, like dimensions
});

Check out Github repository for more documentation.
